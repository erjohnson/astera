Asset Reference
===============

This guide overviews how to manage assets using different methods within Astera. Please keep in mind that this is both still an early versiono of astera, as well as under construction.

Function Ordering
^^^^^^^^^^^^^^^^^

The only order specific functions for the asset header is the initialization of asset maps.

1. ``asset_map_create_zip`` or ``asset_map_create_pak``
2. ``asset_map_get``
3. ``asset_map_free``

Macros
^^^^^^

None currently.
